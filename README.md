** 2. Object Detection and Tracking **

# ** Description:**
Real-time object detection and tracking using a webcam.

Detects objects using YOLOv8

Tracks objects across frames with Deep SORT

Displays bounding boxes, labels, and tracking IDs

# ** Technologies Used: **

1. Python

2. OpenCV

3. Ultralytics YOLOv8

4. Deep SORT

How to Run:
```bash
python "Object Detection and Tracking .py"
```

Note: Make sure to install dependencies:
```bash
pip install opencv-python ultralytics deep_sort_realtime torch torchvision
```
